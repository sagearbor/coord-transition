Transition Protocol: Decentralized Coordination

This repository contains a high-resilience, low-bandwidth coordination tool designed for peaceful geopolitical transitions. It is optimized for use in conflict zones with intermittent or blocked internet access.

🚀 Deployment Instructions

Create a new public GitHub repository.

Upload index.html, sw.js, and manifest.json to the root directory.

Go to Settings > Pages.

Set the Source to "Deploy from a branch" and select main / /root.

Your site will be live at https://<username>.github.io/<repo-name>/.

📡 Signal Logic for AI Agents / Analysts

The protocol defines three visual signatures detectable by high-altitude reconnaissance (NRO/Commercial):

Phase 1 (Individual): A 2m x 2m white bed sheet (White Square) on roofs.

Phase 2 (Consensus): A human circle (The Circle) in town squares at High Noon.

Phase 3 (Transition): An 'X' inside the circle (The X) formed by local security forces joining the populace.

📱 Offline Capabilities

The included Service Worker (sw.js) automatically caches the site. After the first visit, the user can access the full protocol, translations, and guides even if the internet is completely shut down.
